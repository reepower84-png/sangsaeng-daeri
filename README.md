# 상생대리운전 📞 1866-1240

전국 24시간 대리운전 호출 앱 (PWA · 모바일 최적화)

## 기능
- **호출 버튼 3종** — 대리기사 / 탁송기사 / 법인기사 → 누르면 `1866-1240` 자동 전화 연결
- **현재 위치 확인** — 휴대폰 GPS로 지도(OpenStreetMap)에 현재 위치 표시, 카카오맵·네이버·구글 지도 바로 열기, 위치 공유
- **앱 설치(PWA)** — 휴대폰 홈 화면에 앱처럼 설치
- **공유하기** — 카톡 등으로 앱/위치 공유
- 상호 **상생대리운전**, 대표번호 **1866-1240** 화면 표시

## 로컬 실행
```bash
node server.js
# http://localhost:5173
```

## 배포
GitHub Pages (HTTPS) — 위치/설치/공유 기능은 HTTPS에서 동작합니다.

## 파일
| 파일 | 설명 |
|------|------|
| `index.html` | 메인 앱 화면 |

| `generate-icons.js` | 아이콘 생성 스크립트 |
